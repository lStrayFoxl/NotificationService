# Notification-service
Сервис предоставляет API для отправки уведомлений и получения уведомлений пользователя.

## Инструкция по запуску сервиса
- Склонировать репозиторий на локальную машину.
- Выполнить команду 'make setup_dev' для настройки сервиса.
- Запустить приложение командой 'make run_dev'.

## Инструкция по выключению сервиса
- Для выключения приложения можно использовать команду 'make stop'

## Тестирование работы сервиса
Для тестирования проекта можно использовать Postman. Конфигурация коллекции находится в корне проекта(имя файла конфигурации: Notification Service.postman_collection.json).

Коллекция содержит 2 тестовых сценария:
- Отправка уведомления указанным пользователям по указанному каналу(sms, email) - Send Notifications;
- Получение списка уведомлений указного пользователя - Get User Notifications;

## Используемый стек технологий
- PHP(Laravel)
- PostgreSQL
- RabbitMQ
- Redis
- Docker

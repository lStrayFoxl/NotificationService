# Notification-service
Сервис предоставляет API для отправки уведомлений и получения уведомлений пользователя.

## Инструкция по запуску сервиса
- Склонировать репозиторий на локальную машину.
- Выполнить команду 'make setup_dev' для настройки сервиса.
- Запустить приложение командой 'make run_dev'.

## Инструкция по выключению сервиса
- Для выключения приложения можно использовать команду 'make stop'

## Используемый стек технологий
- PHP(Laravel)
- PostgreSQL
- RabbitMQ
- Redis
- Docker
